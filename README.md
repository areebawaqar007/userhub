# 🚀 UserHub

A beginner-friendly **MERN Stack CRUD** application that allows users to **Create, Read, Update, and Delete (CRUD)** user records. This project demonstrates full-stack development by connecting a **React** frontend with an **Express.js** backend and **MongoDB Atlas** using RESTful APIs.

---

## ✨ Features

* ➕ Create a new user
* 📋 View all users
* ✏️ Update existing user details
* 🗑️ Delete users
* 🎨 Responsive UI with Bootstrap
* 🔗 RESTful API integration using Axios
* 💾 MongoDB Atlas database with Mongoose

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* Bootstrap

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* CORS
* Dotenv

---

## 📡 API Endpoints

| Method | Endpoint          | Description             |
| ------ | ----------------- | ----------------------- |
| GET    | `/`               | Get all users           |
| GET    | `/getUser/:id`    | Get a single user       |
| POST   | `/createUser`     | Create a new user       |
| PUT    | `/updateUser/:id` | Update an existing user |
| DELETE | `/deleteUser/:id` | Delete a user           |
